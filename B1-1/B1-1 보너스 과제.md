## 2. 보너스 1 - 나만의 GPT 배포

### GPT 이름
Meeting Assistant Pro

### 역할
10년 경력 프로젝트 매니저

### 주요 기능
- 회의록 요약
- Action Item 추출
- 리스크 분석
- 임원 보고서 작성

### 재사용 입력 템플릿

```text
[회의록 원문]

[원하는 결과]
결정사항
Action Items
리스크
후속 일정
추가 확인 필요 사항

[출력 조건]
- 표 형식 사용
- 담당자/작업/기한 포함
- 원문에 없는 내용 단정 금지
- 모호한 내용은 확인 필요 표시
```

## 3. 보너스 2 - 업무 결과 시각화

### 시각화 대상 데이터

| 구분 | 내용 |
|--------|--------|
| 프로젝트 목표 | 7월 말 MVP 개발 완료 |
| 주요 결정사항 | 회원가입·로그인 우선 개발, PG사 토스페이먼츠 확정, 강의 영상 업로드 MVP 포함 |
| 주요 리스크 | 관리자 페이지 담당자 미정, 일정 부담 증가, 기한 미확정 |
| 후속 액션 | 담당자 지정, 일정 수립, 기한 확정, 임원 보고 자료 검토 |

### 이미지 생성 프롬프트

```text
온라인 교육 플랫폼 MVP 개발 현황을 한눈에 볼 수 있는 한국어 인포그래픽을 만들어줘.
프로젝트 개요, 주요 결정사항, Action Items, 주요 리스크, 개발 진행 현황, 후속 일정으로 구성.
파란색/초록색 업무 보고서 스타일.
16:9 비율.
```
